# CloudCVfy Webapp

Current Status: A basic skeleton of app with some components, actions, reducers and their tests in place in place.

## Build instructions

### Dev

_Hot reloading, immutability check in reducers,  is supported._

1. `npm install`
2. `npm start -s`

### Production

1. `npm install`
2. `npm run build -s`
